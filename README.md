# Sistema de Pedidos API

API REST construída com Spring Boot.

## 🚀 Tecnologias
- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- JWT Authentication

## 📌 Endpoints

### Produtos
GET /api/products

### Pedidos
GET /api/orders  
POST /api/orders  

### Auth
POST /auth/login

## 🔐 Autenticação
JWT (Bearer Token)

## ▶️ Como rodar

```bash
mvn spring-boot:run
```
